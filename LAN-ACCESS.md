# 小贷台账局域网访问说明

## 1. 启动服务

在 `D:\ai` 里运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\start_mobile_server.ps1
```

启动后窗口里会显示几条可访问网址，例如：

```text
http://127.0.0.1:8080/
http://192.168.1.25:8080/
```

## 2. 给其他电脑访问

- 你自己的电脑打开 `http://127.0.0.1:8080/`
- 同一办公室或同一路由器下的其他电脑、手机，打开窗口里显示的 `http://你的局域网IP:8080/`
- 只要服务器这台电脑不关机，并且这个 PowerShell 窗口不关闭，大家就能共用同一份数据

## 3. 数据位置

- 实时数据文件：`D:\ai\shared_lender_data.json`
- 自动备份目录：`D:\ai\shared_backups`

## 4. 重要提醒

- Windows 防火墙如果弹窗，请允许 `PowerShell` 或当前脚本通过专用网络
- 如果其他电脑打不开，先确认几台设备在同一个局域网
- 如果你想要“外网也能访问”的正式网址，还需要再部署到云服务器或做端口映射
